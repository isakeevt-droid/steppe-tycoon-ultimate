# Steppe Tycoon v8.0

Структура проекта:
- `backend/` — FastAPI backend
- `frontend/` — HTML/CSS/JS
- `requirements.txt`

## Запуск

```bash
pip install -r requirements.txt
uvicorn backend.main:app --reload
```

Открыть:
- `http://127.0.0.1:8000/`

## Что обновлено
- ручная продажа по числу
- продажа товаров прямо во вкладке «Производство»
- убрана вкладка «Все», оставлены только «Производящие» и «Перерабатывающие»
- цены охраны показаны в интерфейсе караванов
- подписи автоматики: авто-продажа / авто-переработка
- ускорена логика авто-продажи на backend

## Важно
Если после обновления старая база даёт странное поведение, удалите `data/steppe_tycoon.db` и запустите игру заново.
